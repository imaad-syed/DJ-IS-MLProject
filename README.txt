How to Run:

1. Create two files called train.mv and test.mv, storing the training and test data respectively.
2. Move them to where you want to run the python file.
3. Run the file with "python XGBoost.py"
4. The program will return a prediction file for the test data as well as output for the accuracy of the randomly seperated training and validation data.
